Resources

Here you can find a collection of useful programs used in dynamics and other fun things too.

  • Various programs created by Alden Walker, PhD.

    They are amazing programs and easy to use. I have been playing with the schottky program to study the boundary of the connectedness locus of a certain family of IFS.

  • Complex Trees A CDF-based website that guides its visitors around the forest of symmetric fractal trees.

    Bernat Espigulé Pons has mapped the generalized families of planar self-contacting symmetric fractal trees.
    (It's a work in progress)

  • Mandel created by Wolf Jung, PhD.

    Great interactive program to study the dynamics of Quadratic, Cubic, and Quartic polynomials; Rational, Trascendental, Newton, and Non-Analytic mappings.
    Available for Linux, Windows, and Mac.

  • FractalStream created by Matthew Nooan, PhD.

    One of the best programs. It is simple to use and it allows the user to write his own scripts. It also supports real dynamics.
    Available for Mac. Here is its GitHub page.

  • Dynamics Explorer created by Brian Boyd, PhD.

    A program designed to let users explore dynamical systems, particularly focused on enabling mathematicians to improve the efficiency and quality of their research in the field of complex dynamics.
    It allows the user to write his own scripts (called profiles).

    Available for Linux, Windows, and Mac.

  • Various programs created by Curtis McMullen, PhD.

    Available for Mac.

  • Java applets to explore Chaos, Fractals, and compute the parameter planes and Julia sets for complex functions.

    You might want to use IE for these. Uugh!!!

  • Magic Tile created by Roice Nelson.

    Fun program that lets you play with topological twisty puzzles (!!) such as the Rubik's Klein Bottle; Rubik's Torus, and many more!
    Burkard Polster (aka Mathologer) has a couple of videos (Vid1, Vid2) describing this program.
