Resources

Here you can find a collection of useful programs used in dynamics.

  • Mandel created by Wolf Jung, PhD.

    Great interactive program to study the dynamics of Quadratic, Cubic, and Quartic polynomials; Rational, Trascendental, Newton, and Non-Analytic mappings.
    Available for Linux, Windows, and Mac.

  • FractalStream created by Matthew Nooan, PhD.

    One of the best programs. It is simple to use and it allows the user to write his own scripts. It also supports real dynamics.
    Available for Mac. Here is its GitHub page.

  • Various programs created by Curtis McMullen, PhD.

    Available for Mac.

  • Java applets to explore Chaos, Fractals, and compute the parameter planes and Julia sets for complex functions.
    You might want to use IE for these. Uugh!!!
